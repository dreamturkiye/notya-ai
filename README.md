# Notya AI 🇹🇷

> **Konuş, Biz Yazalım** — Türkiye'nin Evrensel Profesyonel AI Not Asistanı

## Desteklenen Meslekler
- 🏥 Doktorlar (14 uzmanlık)
- ⚖️ Avukatlar (6 hukuk alanı)
- 🧠 Psikologlar/Terapistler (8 terapi türü)
- 📊 Mali Müşavirler (6 hizmet türü)
- 👥 İnsan Kaynakları Uzmanları
- 🏠 Emlak Danışmanları
- 🛡️ Sigorta Acenteleri
- 📚 Eğitim Danışmanları
- 💼 Yöneticiler

## Tech Stack
- **Next.js 14** — App Router
- **Supabase** — Database + Auth (EU Frankfurt)
- **Claude API** — AI note generation
- **Deepgram** — Real-time Turkish transcription
- **iyzico** — Turkish payment processing
- **WhatsApp Business API** — Note delivery

## Uyumluluk
- ✅ KVKK (Kişisel Verilerin Korunması Kanunu)
- ✅ Sağlık Bakanlığı veri düzenlemeleri
- ✅ AES-256-GCM şifreleme
- ✅ EU data residency (Frankfurt)

npm install
